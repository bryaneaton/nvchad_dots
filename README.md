
## Install NVChad
`git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1` 

## Install Custom Config with Copilot
`git clone https://github.com/bryaneaton/nvchad_dots.git ~/.config/nvim/lua/custom --depth 1 && nvim`
